===========================================================
                    CULT TRANSLATOR — README
===========================================================

>>> OVERVIEW
Cult Translator is a next-generation AI translation plugin 
for Adobe After Effects that makes your translations easy, 
fast, and precise. It supports 100 languages and allows you 
to add context so every translation reflects the true intent 
of your design.

-----------------------------------------------------------

>>> INSTALLATION
1) Open Adobe After Effects.
2) Go to File → Scripts → Install Script UI Panel…
3) Select the file "CultTranslator.jsx".
4) Restart After Effects.
5) Open it under Window → Cult Translator.
6) Enter your license key to activate, or start your 7-day trial.
